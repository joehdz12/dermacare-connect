# DermaCare Connect

Aplicación web para la clínica dermatológica del Dr. Alvaro J. Ramos.

## Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo estará disponible en `http://localhost:8080`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
